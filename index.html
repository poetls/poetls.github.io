Visual Report Generation

The details of each visualisation report can be presented to the user as an example. Within this website, users can access various gait parameters, view the gait status of our subjects and compare the subjects' status with reference levels. The visualised gait samples are used for the implementation of gait data visualisation, converting raw unreadable data into visual graphs so that doctors can easily interact with them.

Gait Analysis

The gait analysis page shows the stages of work for the user. The visualisation report is uploaded and processed and displayed to the user, who can assess the subject's gait condition and give recommendations and comments. Once the assessment is complete, they can export a report including the visualisation and evaluation as a finalised prescription.
